This scraper is SEPARATE from your main transport network project.

To integrate the scraped fare data with your existing network:

1. Load both datasets:
   ```python
   import pandas as pd
   
   # Your existing network
   nodes = pd.read_csv('../nodes.csv')
   edges = pd.read_csv('../edges.csv')
   
   # Scraped metro fares
   fares = pd.read_csv('data/metro_fares_clean.csv')
   ```

2. Merge metro edges with fare data:
   ```python
   metro_edges = edges[edges['mode'] == 'metro']
   
   enriched = metro_edges.merge(
       fares,
       left_on=['from_node_name', 'to_node_name'],
       right_on=['from_station', 'to_station'],
       how='left'
   )
   ```

3. Update your network with actual fares:
   ```python
   edges.loc[edges['mode'] == 'metro', 'cost'] = enriched['fare_rs']
   edges.to_csv('../edges_with_fares.csv', index=False)
   ```
